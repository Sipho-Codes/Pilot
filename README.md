# Pilot
Simple DIY marketplace landing page built with HTML and CSS.

## Project goal
Create a lightweight single-page site to showcase a do-it-yourself marketplace concept for makers, craft sellers, and small brand storefronts.

## Features
- Hero section with clear marketplace positioning
- Product collection preview
- Seller-focused benefits and setup steps
- Responsive layout for mobile and desktop

## Files
- `index.html` — page structure and content
- `styles.css` — styling for layout, colors, cards, and responsive design

## Usage
Open `index.html` in your browser to preview the page locally.
